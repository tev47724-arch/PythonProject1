# PSU Harrisburg AI Chatbot

This is an AI chatbot project built using Python, Streamlit, and OpenRouter.

The chatbot answers questions about Penn State Harrisburg by reading information from PSU webpages such as academics, admissions, campus life, and academic calendars.

## Features

- AI-powered chatbot
- Reads PSU Harrisburg website content
- Answers student questions
- Simple Streamlit web interface
- Uses OpenRouter with Gemini model

## Technologies Used

- Python
- Streamlit
- BeautifulSoup
- Requests
- LangChain
- OpenRouter API
